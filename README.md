# Proyecto Final - Programación

## Descripción
Este proyecto desarrolla una solución en Python para gestionar el inventario de una empresa.

## Funcionalidades
- Almacena los productos en una matriz.
- Calcula automáticamente la cantidad de unidades que deben solicitarse.
- Genera un reporte con la información del inventario.
